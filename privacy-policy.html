隱私政策

最後更新日期：2025 年 11 月

核心承諾：我們僅使用您授權的最小必要權限，所有資料僅用於提供應用功能，不會儲存或分享給任何第三方。

1. 適用範圍

本隱私政策適用於 ThinkerCafe 所開發的 Threads Tool 應用程式（以下簡稱「本應用」）。使用本應用即表示您同意本隱私政策的內容。

2. 資料收集與使用

本應用僅會在您明確授權的情況下，透過 Threads API 進行以下操作：

  • 讀取內容：讀取您授權範圍內的 Threads 貼文、個人資料等公開資訊
  • 發佈內容：根據您的指示，代表您發佈或管理 Threads 貼文
  • 帳號資訊：取得基本帳號資訊（如使用者名稱、個人檔案）以提供服務功能

3. 資料處理原則

我們嚴格遵守以下原則：

  • 最小化原則：僅請求提供服務所必需的最低權限
  • 即時處理：所有資料僅在執行您的指令時即時處理，不進行長期儲存
  • 不分享原則：您的資料絕不會被出售、出租或分享給任何第三方
  • 透明化：所有資料使用目的均會明確告知

4. 資料安全

我們採取業界標準的安全措施保護您的資料：

  • 使用 HTTPS 加密傳輸協定
  • 遵循 Threads API 官方安全指引
  • 定期進行安全性審查與更新

5. 您的權利

您擁有以下權利：

  • 授權管理：您可以隨時在 Threads 設定中撤銷本應用的授權
  • 資料查詢：您可以要求查詢我們處理您資料的方式
  • 刪除要求：您可以要求刪除任何暫存的資料

6. Cookie 與追蹤技術

本應用不使用 Cookie 或任何追蹤技術來收集您的個人資訊。

7. 第三方服務

本應用使用 Meta（Facebook/Instagram）提供的 Threads API 服務。您的資料處理同時受到 Meta 隱私政策的約束。

8. 兒童隱私

本應用不針對 13 歲以下兒童設計，也不會故意收集兒童的個人資訊。

9. 政策更新

我們可能會不定期更新本隱私政策。重大變更將透過應用內通知或電子郵件告知。建議您定期查看本頁面以了解最新政策。

10. 適用法律

本隱私政策的解釋與適用，以及因使用本應用所生之爭議，均應依照中華民國法律處理。我們同時遵守 GDPR（歐盟一般資料保護規範）及其他適用的資料保護法規。
