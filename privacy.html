This Privacy Policy applies to the Threads Tool application developed by ThinkerCafe (hereinafter "the Application"). By using the Application, you agree to the contents of this Privacy Policy.

2. 資料收集與使用Data Collection and Use

本應用僅會在您明確授權的情況下，透過 Threads API 進行以下操作：

The Application will only perform the following operations through the Threads API with your explicit authorization:

  • 讀取內容 Read Content: 讀取您授權範圍內的 Threads 貼文、個人資料等公開資訊
  • Read your authorized Threads posts, profile, and other public information
  • 發佈內容 Publish Content: 根據您的指示，代表您發佈或管理 Threads 貼文
  • Publish or manage Threads posts on your behalf according to your instructions
  • 帳號資訊 Account Information: 取得基本帳號資訊（如使用者名稱、個人檔案）以提供服務功能
  • Obtain basic account information (such as username, profile) to provide service functions

3. 資料處理原則Data Processing Principles

我們嚴格遵守以下原則：

We strictly adhere to the following principles:

  • 最小化原則 Minimization: 僅請求提供服務所必需的最低權限
  • Only request the minimum permissions necessary to provide the service
  • 即時處理 Real-time Processing: 所有資料僅在執行您的指令時即時處理，不進行長期儲存
  • All data is processed in real-time only when executing your commands, with no long-term storage
  • 不分享原則 No Sharing: 您的資料絕不會被出售、出租或分享給任何第三方
  • Your data will never be sold, rented, or shared with any third party
  • 透明化 Transparency: 所有資料使用目的均會明確告知
  • All data usage purposes will be clearly communicated

4. 資料安全Data Security

我們採取業界標準的安全措施保護您的資料：

We employ industry-standard security measures to protect your data:

  • 使用 HTTPS 加密傳輸協定 Use HTTPS encrypted transmission protocol
  • 遵循 Threads API 官方安全指引 Follow official Threads API security guidelines
  • 定期進行安全性審查與更新 Regular security reviews and updates

You have the following rights:
